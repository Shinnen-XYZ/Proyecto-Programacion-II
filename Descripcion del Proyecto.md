## Descripción del Proyecto

Este proyecto fue desarrollado como parte del curso **Programación II**, y consiste en un sistema para administrar estudiantes usando una aplicación web conectada a un backend hecho con **Spring Boot**.

La idea general es poder **crear, editar, listar y eliminar estudiantes**, manejando sus datos básicos como nombre, correo, teléfono e idioma. La información se guarda en **memoria principal**, por lo que no utiliza base de datos.

El diseño del frontend se realizó con **HTML, CSS y JavaScript**.
